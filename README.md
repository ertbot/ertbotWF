#ERTBOT WORKFLOW

For www.ertsybot.com

##How to use this?
Make sure you have Node & NPM just type 
````
node -v
npm -v
````
also make sure you have a grunt-cli
````
npm install -g gulp-cli
````
Just clone this repository
````
````
cd ertbotWF
npm install
````
and lastly use gulp to start working
````
gulp
````
